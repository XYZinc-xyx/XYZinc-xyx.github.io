C++STL常见容器

发表于 2022-03-28 更新于 2022-04-22 分类于 语言
C++STL部分容器的一些操作

iterator 迭代器

迭代器是一种检查容器内元素并遍历元素的数据类型

  • 声明方式：容器类型<数据类型>::iterator 名称；

stack 栈

  • 头文件：#include <stack>

  • 声明方式：stack<类型>名称;

vector 动态数组

  • 头文件：#include <vector>

  • 声明方式：vector<类型>名称;

queue 队列

  • 头文件：#include <queue>

  • 声明方式：queue<类型>名称；

优先队列 讲解及模板

map 映射

  • 头文件：#include <map>

  • 声明方式：map<类型1, 映射后类型2>名称;

  • map特性：map会按照键的顺序从小到大自动排序

set 集合

  • 头文件：#include <set>

  • 声明方式：set<类型>名称;

  • set特性：集合中的元素不重复，且从小大大自动排序
